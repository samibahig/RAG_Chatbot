

# RAG_Chatbot

Chatbot Support Client utilisant LangChain et RAG pour répondre aux questions basées sur des documents.

## Structure
- `app.py` : code Streamlit du chatbot
- `data/support_doc.txt` : documents de support
- `requirements.txt` : packages Python nécessaires

## Déploiement
- Déployer sur Streamlit Cloud :
```bash
pip install -r requirements.txt
streamlit run app.py
